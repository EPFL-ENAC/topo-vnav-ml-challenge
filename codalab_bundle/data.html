Folder and data description 

 

Folder name + content description

Exemple 

rgb_synth


synthetics images

Format : png 

Dimensions : 720 x 480 px

rgb_real


real images 

Format : png 

Dimensions : 720 x 480 px

poses


Ground truth matrix 6D position.

Format : text file

 

Where the . represent the element of the 3 rotation matrix of heading-pitch-roll NED in ECEF coordinate system.

x,y,z are in the local ECEF coordinate system.

This matrix can be transformed in [lat, lon, h, yaw, pitch, roll] using poses_2_csv.py 

calibration 

focal length

  

depth :

z-buffer depth (computed following [5])

Format: Torch object

 

init 

scene coordinate (XYZ per pixel)

Format: Torch object

normal

Surface normal is computed using Open3D.

Format: Torch object

 

semantics 

The semantic labels have seven classes: sky, ground, vegetation, building, water, bridge, and others.

Format: Torch object
