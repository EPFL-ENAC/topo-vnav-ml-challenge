Submission Format

To submit your results to the leaderboard you must construct a submission zip file containing a CSV file.
This file must have the follwing columns (without column names) :
  • Column 1 : X coodinate
  • Column 2 : Y coodinate
  • Column 3 : Altitudes
  • Column 4 : Azimuth
  • Column 5 : Tilt
  • Column 6 : Roll
Coordinates are in a local coordinate system.

Evaluation Criterias

The median error on the coordinates is calculated. Based on this value, the score is established as follow :
  • if the median = 0 m → score on coordinates = 100
  • if the median > 100 m → score on coordinates = 0
  • if 0 < the median < 100 m → score on coordinates = 100 - the median
The median error on the angles is calculated. Based on this value, the score is established as follow :
  • score on angles = (360 - median) * 100 / 360

overall score = score on coordinates * 0.75 + score on angles * 0.25
